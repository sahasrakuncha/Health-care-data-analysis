# Health-care-data-analysis
Healthcare systems generate a significant amount of data from patient records, laboratory reports, and medical monitoring devices.
Managing and analyzing this large volume of data is important for improving healthcare services and supporting medical decision making.
                 This project focuses on Healthcare Data Analytics using Big Data technologies and Machine Learning techniques to analyze healthcare datasets. Apache Spark is used for efficient processing of large-scale healthcare data, while machine learning models analyze patient parameters such as age, blood pressure, sugar levels, and cholesterol to identify potential health risks.
                 The system also provides data visualization through a dashboard to present meaningful insights from the analyzed data. This approach enables healthcare professionals to better understand patient conditions and supports improved healthcare management.
